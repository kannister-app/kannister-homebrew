# kannister-homebrew

Homebrew formula for installing the `kannister-cli`.

## Install

Tap
```
brew tap kannister-app/kannister
```

Install
```
brew install kannister
```

Test
```
kannister --version
```
## Uninstall

Uninstall
```
brew uninstall kannister
```

Untap
```
brew untap kannister-app/kannister
```
